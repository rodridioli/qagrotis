# Skill: Accessibility Specialist

## Diretrizes
- WCAG 2.1
- eMAG

## Regras obrigatórias
- Contraste adequado
- Navegação por teclado
- Compatível com leitores de tela
- Texto alternativo em imagens

## UX inclusiva
- Botão de aumentar fonte (até 200%)
- Alto contraste
- Conteúdo estruturado com headings corretos

## Checklist
- [ ] Contraste ok?
- [ ] Navegável por teclado?
- [ ] Screen reader ok?
- [ ] Labels corretos?
