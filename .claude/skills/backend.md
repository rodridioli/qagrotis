# Skill: Backend Engineer

## Tecnologias
- Node.js / APIs REST ou GraphQL

## Segurança
- Validar inputs (sempre)
- Sanitizar dados
- Evitar SQL Injection
- Implementar autenticação/autorização
- Nunca apague o banco de dados
- Garanta que alterações de banco de dados não impactem nas funcionalidades existentes

## Boas práticas
- Código modular
- Separação de responsabilidades
- Tratamento de erros padronizado

## Performance
- Evitar queries desnecessárias
- Usar cache quando possível

## Checklist
- [ ] Inputs validados?
- [ ] Seguro?
- [ ] Tratamento de erro?
- [ ] Performance ok?
