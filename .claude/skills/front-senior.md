# ⚛️ Skill: Front-end Senior (Execução)

## Regras obrigatórias
- Usar 100% Design System (tokens) :contentReference[oaicite:4]{index=4}
- Proibido hardcode :contentReference[oaicite:5]{index=5}
- Mobile-first
- Acessível (aria, semântica)

## Ações
1. Criar/usar componentes do DS
2. Se não existir:
   - Criar componente
   - Adicionar no Storybook
3. Garantir:
   - Responsividade
   - Performance
   - Estados definidos pelo UX

## Performance
- Lazy loading :contentReference[oaicite:6]{index=6}
- Memoização
- Code splitting

## UX avançado (OBRIGATÓRIO)
- Skeleton loading (evitar flicker)
- Debounce em inputs
- Cancelamento de requests (AbortController)
- Estados vazios bem definidos

## Acessibilidade avançada
- Focus management (focus trap, focus visible)
- Navegação completa por teclado
- Feedback para screen readers
- Não depender apenas de cor

## Tratamento de erro avançado
- Mensagens amigáveis
- Retry de ações
- Fallback UI

## Saída
- Código React/Next.js
- Componentes reutilizáveis
- Storybook atualizado

## Checklist
- [ ] Sem hardcode?
- [ ] Usa tokens?
- [ ] Responsivo?
- [ ] Acessível?
- [ ] Está no Storybook?
