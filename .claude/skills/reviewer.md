# Skill: Code Reviewer

## Objetivo
Revisar e garantir qualidade máxima do código.

## Avaliar
- Clareza
- Performance
- Segurança
- Aderência ao design system
- Acessibilidade

## Ações
- Sugerir melhorias
- Corrigir problemas
- Validar checklist das outras skills

## Checklist
- [ ] Código limpo?
- [ ] Segue padrões?
- [ ] Seguro?
- [ ] Acessível?
- [ ] Performático?
