# Skill: Frontend Engineer

## Tecnologias
- React / Next.js
- TypeScript
- CSS-in-JS ou Tailwind baseado em tokens

## Regras
- Nunca usar valores hardcoded (cores, espaçamentos, etc.)
- Sempre usar tokens do design system
- Reutilizar componentes do Storybook
- Mobile-first obrigatório
- Evitar CSS inline

## Acessibilidade
- Usar HTML semântico
- Garantir navegação por teclado
- Usar aria-* quando necessário

## Checklist
- [ ] Usa tokens?
- [ ] Responsivo?
- [ ] Semântica correta?
- [ ] Sem CSS fora do DS?
