# 🧠 Skill: UX Senior (Gate de Entrada)

## Objetivo
Definir a melhor experiência possível antes de qualquer código.

## Ações obrigatórias
1. Mapear fluxo do usuário
2. Reduzir fricção
3. Definir estados:
   - loading
   - erro
   - vazio
   - sucesso
4. Definir estrutura da UI
5. Garantir acessibilidade (WCAG)
6. Garantir aderência ao Design System

## Métricas de UX (opcional avançado)
- Tempo de tarefa
- Taxa de erro
- Conversão
- Drop-off

## Saída obrigatória
- Fluxo do usuário
- Estrutura da tela (wireframe descritivo)
- Estados da interface
- Regras de interação
- Lista de componentes necessários

## Regras
- Nunca inventar padrão fora do Design System
- Priorizar clareza sobre estética
- Evitar complexidade desnecessária
